# DWH-41
В данном репозитории находятся sql-скрипты для выполнения командной работы группы 41 по курсу "Хранилища данных" по созданию хранилища данных для аэропортов со всей необходимой информацией о перелетах, погодных условиях, авиакомпаниях и т.д.
Здесь находятся скрипты для создания таблиц на различных уровнях, для заполнения их данными с изменениями от уровня к уровню.
